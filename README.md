# ⚔️ The Iron Market - Game of Thrones Playing Cards Store

A premium ecommerce store for Game of Thrones themed playing cards, built with Next.js 14 + Tailwind CSS.

## Features

- 🏠 **Home Page** with product grid, search, and house filtering
- 📱 **Product Detail Pages** with features, reviews, and related products
- 🛒 **Shopping Cart** persisted in localStorage
- 📜 **Mock Checkout** with shipping + payment forms
- 🎭 **12 Products** across 7 Great Houses of Westeros
- 🎨 **Dark theme** with gold accents

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- localStorage for cart persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the store.
